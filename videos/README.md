# Видео-файлы для галереи

## Как добавить видео:

### Вариант 1: Локальные файлы
Поместите ваши видео файлы в эту папку с названиями:
- `video1.mp4` - Оформление документов
- `video2.mp4` - Работа водителем C+E
- `video3.mp4` - Маршруты по Европе

### Вариант 2: Google Drive
1. Загрузите видео на Google Drive
2. Сделайте файл доступным по ссылке (Настройки доступа → Доступен всем, у кого есть ссылка)
3. Скопируйте ID файла из ссылки (часть между `/d/` и `/view`)
4. В index.html замените `./videos/video1.mp4` на:
   ```
   https://drive.google.com/uc?export=download&id=YOUR_FILE_ID
   ```

### Вариант 3: GitHub
1. Добавьте видео файлы в эту папку
2. Закоммитьте и запушьте в репозиторий
3. Vercel автоматически подхватит видео при деплое

## Рекомендуемые параметры видео:
- Формат: MP4 (H.264)
- Разрешение: 1280x720 или 1920x1080
- Размер: не более 50MB на видео
- Длительность: 30-60 секунд

## Текущие плейсхолдеры:
Пока видео не добавлены, будут отображаться SVG-заглушки с подписями.
